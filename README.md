# f002

Formulario HSE-F002 construido con Next.js.

## Estructura

- `app/page.tsx`: orquestador principal del formulario
- `app/secciones-formulario.tsx`: secciones visuales del formulario
- `app/componentes-formulario.tsx`: componentes reutilizables de campos
- `app/configuracion-formulario.ts`: listas y configuraciones del formulario
- `app/firma-canvas.tsx`: componente de captura de firmas
- `app/globals.css`: estilos globales

## Desarrollo

```bash
npm install
npm run dev
```

## Comandos

```bash
npm run build
npm run lint
```
